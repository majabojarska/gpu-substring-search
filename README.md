# [GPU] Substring Search

## Useful VsCode extensions
* PlantUML (diagrams)
* LaTeX Workshop (with texlive, latexmk from CLI)
* Spellright (docs, uses Hunspell)

## Setup & run

Install dependencies:
```sh
npm install
```

Run dev server:
```sh
npm run serve
```

Build in dev mode:
```sh
npm run build
```

Build in dev and watch:
```sh
npm run build:watch
```

Build for production:
```sh
npm run build:prod
```
